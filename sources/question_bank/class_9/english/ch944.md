# Chapter 944: Letter to the Editor

This unit tests Creative Writing Skills via a formal Letter to the Editor of a newspaper/magazine.
Prompts must be set in realistic, age-appropriate social contexts (e.g. rising use of single-use plastics, need for community libraries, importance of physical activities for children).
Each writing prompt must explicitly define:
- Target Audience (e.g., The Editor of a national daily, local newspaper)
- Purpose (e.g., highlighting a civic issue, suggesting solutions)
- Format (Formal Letter structure: sender/receiver addresses, date, subject, salutation, body paragraphs, complimentary close)
- Word Limit (100-120 words)
- Rubric / Evaluation Criteria (e.g. Format: 1 mark, Content: 2 marks, Expression & Accuracy: 2 marks, totaling 5 marks).

Explanations must include a high-quality model response and a detailed marking rubric.
